## 📜 Resumo do Projeto

Esta API RESTful é uma aplicação de gestão de receitas veganas, desenvolvida para demonstrar proficiência em desenvolvimento Back-end e a entrega completa de um produto na nuvem (Cloud Computing).

* **O que faz:** Fornece endpoints para CRUD (Criar, Ler, Atualizar, Deletar) de receitas.
* **Back-end:** Node.js com o framework Express.
* **Banco de Dados:** MongoDB Atlas (NoSQL) para persistência de dados.

---

## 🛠️ Stack Tecnológica e Deploy (DevOps)

O diferencial deste projeto é a utilização de **Infraestrutura como Código (IaC)** para automatizar todo o deploy do Back-end, garantindo que o ambiente seja replicável e de baixo custo (Serverless).

| Categoria | Tecnologia | Uso no Projeto |
| :--- | :--- | :--- |
| **Código** | Node.js (Express) | Lógica da aplicação e rotas. |
| **Banco de Dados** | MongoDB Atlas | Hospedagem externa e segura dos dados. |
| **Containerização** | Docker | Empacotamento da aplicação para execução uniforme. |
| **Infraestrutura** | **Terraform (IaC)** | Gerenciamento e provisionamento da VPC, Security Group, e ECS Fargate. |
| **Orquestração** | **AWS ECS Fargate** | Execução Serverless do container Docker. |

O deploy foi validado através de um teste de requisição HTTP, que retornou o código **`200 OK`**, confirmando a comunicação entre o **Node.js** na AWS e o **MongoDB Atlas**.
